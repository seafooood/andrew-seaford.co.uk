# Secondary Storage

Secondary storage provides persistent data storage that is retained when the computer is powered off. In GCSE Computer Science, you need to understand different types of secondary storage including magnetic (hard drives), solid state (SSDs), and optical (CDs/DVDs), comparing their speed, capacity, and cost.
