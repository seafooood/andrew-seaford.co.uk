# Week 06 Experiment: Restore GCSE Revision Content

## Hypothesis

The primary driver of zero clicks despite hundreds of impressions is missing GCSE revision content that existed on the old WordPress site but was never migrated to the new Docusaurus site.

Google still indexes the old revision section URL and users are actively searching for vacuum forming content, but there are no pages to land on. Adding the revision content — with proper redirects from the old WordPress PDF URLs — will recover indexing and generate real clicks.

---

## Background

GSC shows 927 impressions for "vacuum forming" with 0 clicks. The four currently indexed pages are all from the old WordPress site:

- `http://www.andrew-seaford.co.uk/revision/gsce-computer-science/` — Google remembers the old revision section
- Three PDFs at `/wp-content/uploads/2021/03/` paths — no longer accessible on the new site

The fix is to restore this content on the Docusaurus site and redirect the old URLs.

## Why Markdown Wrapper Pages?

Yes, each PDF needs a Markdown wrapper page. Reasons:

1. **Google cannot index raw PDFs effectively** — a wrapper page gives the content a crawlable URL with proper metadata
2. **SEO frontmatter** — title, description, and keywords can be set on the wrapper page, not on a PDF
3. **Internal links** — the wrapper can link to related revision topics, passing link equity
4. **User experience** — visitors land on a page explaining the topic before downloading, rather than a raw PDF

---

## Redirect Strategy

Add entries to `vercel.json` (in the existing `redirects` array) for each old WordPress PDF URL. This preserves any link equity from the indexed PDFs and ensures old bookmarks and search results still work.

```json
{
  "source": "/wp-content/uploads/2021/03/Pulleys-and-Belts.pdf",
  "destination": "/revision/gcse-technology/pulleys-and-belts",
  "permanent": true
},
{
  "source": "/wp-content/uploads/2021/03/Smart-Materials.pdf",
  "destination": "/revision/gcse-technology/smart-materials",
  "permanent": true
},
{
  "source": "/wp-content/uploads/2021/03/The-Six-Rs.pdf",
  "destination": "/revision/gcse-technology/the-six-rs",
  "permanent": true
}
```

Additional redirects should be added as more PDFs are identified from the old site.
