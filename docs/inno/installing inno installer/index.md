Detailed below is the procedure for installing Inno.

- Download the setup file. I will be installing version 5.5.5 http://www.jrsoftware.org/download.php/is.exe
- Select your language from the combo box and then click ok[![inno1](images/inno1-300x171.png)](images/inno1.png)
- Click the Next button on the welcome screen.[![inno2](images/inno2-300x249.png)](images/inno2.png)

- Select Accept and then click the next button[![inno3](images/inno3-300x249.png)](images/inno3.png)

- Change the install location if required and then click the next button.[![inno4](images/inno4-300x250.png)](images/inno4.png)

- Change the start menu icon location if required and then click the next button.[![inno5](images/inno5-300x253.png)](images/inno5.png)

- Click Next button to to install the preprocessor. [![inno6](images/inno6-300x251.png)](images/inno6.png)

- Enable the additional settings if required and then click the next button.[![inno7](images/inno7-300x257.png)](images/inno7.png)

- Click the install button to start the installation.[![inno8](images/inno8-300x255.png)](images/inno8.png)
- Click the finish button to complete the install.[![inno9](images/inno9-300x252.png)](images/inno9.png)
